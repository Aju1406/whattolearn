All Shortcut keys in windows of all version


ctrl + A  = to select all
ctrl + B  = bold in word,excel powerpoint
ctrl + C  = Copy selected text,image and etc
ctrl + E  = Alignment in word,excel,powerpoint.
ctrl + F  = Find option.
ctrl + G  = find and replace Option.
ctrl + H  = History in Chrome or web_browser
ctrl + I  = italic the word ...
ctrl + J  = justify the word in office
ctrl + K  = Insert link
ctrl + M  = to add slide in powerpoint.
ctrl + N  = new page in office
ctrl + O  = to open file in office
ctrl + P  = to print
ctrl + R  = to refresh the page
ctrl + S  = to save the page
ctrl + U  = to underline the word or sentence
ctrl + V  = to paste the copy cut content
ctrl + X  = to cut the content
ctrl + Y  = redo option
ctrl + Z  = to undo option
